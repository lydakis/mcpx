## Install / Update

Manual binary:

```bash
# Example for macOS arm64
curl -L -o mcpx https://<release-url>/mcpx-darwin-arm64
chmod +x mcpx
mv mcpx /usr/local/bin/mcpx
```

Verify:

```bash
mcpx --version
```

## Checksums

Use `SHA256SUMS` from the release artifacts:

```bash
shasum -a 256 -c SHA256SUMS
```
